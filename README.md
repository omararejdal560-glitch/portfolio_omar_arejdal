# Portfolio Omar Arejdal

## Description du projet

Ce projet est un portfolio interactif développé en HTML, CSS et JavaScript.
L’objectif est de présenter mon profil, mes compétences ainsi que quelques projets réalisés durant ma formation.

Le site contient plusieurs fonctionnalités interactives et une interface responsive adaptée aux ordinateurs et aux appareils mobiles.

---

## Fonctionnalités principales

* Navigation dynamique
* Mode sombre (Dark Mode)
* Affichage dynamique des projets
* Bouton “Details” pour afficher plus d’informations
* Formulaire de contact avec validation
* Chargement dynamique de données avec Fetch API
* Responsive Design

---

## Technologies utilisées

* HTML5
* CSS3
* JavaScript
* Fetch API
* Async / Await
* GitHub Pages
